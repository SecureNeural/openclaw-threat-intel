# Findings

Canonical intelligence entries live here.

Use `findings/` for the normalized subject, not for every article or social post that mentions it.

## Categories

- `vulnerabilities/`: code or design flaws in OpenClaw or closely related components
- `exposures/`: insecure deployment patterns and exposed attack surfaces
- `incidents/`: real-world misuse, leakage, or abuse cases
- `malicious-skills/`: weaponized skills, poisoned packages, or ecosystem abuse
