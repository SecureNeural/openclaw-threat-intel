# Confidence Ratings

## High

Use when the claim is supported by primary vendor docs, primary research, direct code evidence, or reproducible technical detail.

## Medium

Use when the claim is plausible and supported by credible commentary or partial evidence, but primary proof is incomplete or indirect.

## Low

Use when the item is mostly anecdotal, promotional, or only weakly evidenced.

## Rule

Confidence is about evidence quality, not about severity.
