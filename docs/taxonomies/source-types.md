# Source Types

## Allowed Types

- `article`
- `advisory`
- `docs`
- `post`
- `repo`

## Guidance

- `article`: vendor research, blogs, reports, long-form writeups
- `advisory`: CVEs, GHSAs, signed feeds, vendor security notices
- `docs`: official hardening or configuration documentation
- `post`: LinkedIn or other short-form commentary
- `repo`: release pages, code repositories, indexes, PoCs
