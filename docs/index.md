# OpenClaw Threat Intel

This repository is the shared source of truth for OpenClaw-related attacks, vulnerabilities, incidents, hardening guidance, and supporting evidence.

## Operating Rules

- Create one canonical entry per security subject.
- Store source material separately from normalized analysis.
- Ingest non-English material in English.
- Treat LinkedIn posts as secondary unless they link to a primary source.
- Do not mark a finding as confirmed unless the evidence supports it.

## Current Coverage

- Exposures: local API exposure and remote abuse patterns
- Incidents: data overreach and unintended publication risk
- Controls: gateway hardening, scanner tooling, and defensive monitoring
- Sources: primary docs, articles, and release material
