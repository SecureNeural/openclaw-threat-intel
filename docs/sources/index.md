# Sources

<div class="section-band">
  <p>
    Source records capture the evidence layer. Every canonical finding or control should point
    to one or more source records with clear provenance and trust level.
  </p>
</div>

<div class="panel-grid">
  <div class="intel-card">

  </div>
  <div class="intel-card">
    <div class="intel-eyebrow">Docs</div>
    <h3>Official baselines</h3>
    <p>Product documentation, hardening references, and configuration guidance from the vendor.</p>
  </div>
  <div class="intel-card">
    <div class="intel-eyebrow">Posts</div>
    <h3>Secondary signals</h3>
    <p>Short-form commentary and social references that may point to primary sources or field observations.</p>
  </div>
  <div class="intel-card">
    <div class="intel-eyebrow">Repos</div>

    <p>Repositories, release material, PoCs, and indexes useful for control maturity and implementation evidence.</p>
  </div>
</div>
